# 离线文档库

#### 介绍
爬虫用的是Python3 Scrapy，本文档库会爬取一些主流语言或是框架或是库的API文档，当然如果官网已提供离线文档包，就会直接采用官网的文档包，否则就用爬虫爬。
本文档库包括爬虫的源代码与离线文档包，全体开发者均可免费使用，但因非法使用等原因导致的纠纷本作者不承担任何责任。

#### 安装教程

1.  安装Python3
2.  安装pip3
3.  安装scrapy

#### 使用说明

scrapy crawl <爬虫名>

#### 爬虫说明
目前，爬虫已爬取了Bootstrap、NodeJS与ElectronJS的官方文档
